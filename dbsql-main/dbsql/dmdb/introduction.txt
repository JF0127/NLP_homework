### 表 `XYCS.FLOW_BUSINESS` 简介

#### **表用途**
`FLOW_BUSINESS` 表用于管理和记录与业务流程相关的信息，主要涉及业务类型、编号、名称、描述以及其应用场景（如用户、组织、角色等）。这是一个典型的流程管理系统中的核心表，记录了业务流的关键信息。

---

#### **表结构**

| 字段名                  | 描述                                       |
|--------------------------|--------------------------------------------|
| `ID`                    | 主键，唯一标识每条记录。                    |
| `UNIT_NO`               | 单位编号。                                 |
| `SORT_NO`               | 排序编号。                                 |
| `BUSINESS_CATE`         | 业务类别。                                 |
| `BUSINESS_NO`           | 业务编号，用于区分具体业务。                 |
| `BUSINESS_NAME`         | 业务名称。                                 |
| `IS_DISABLE`            | 是否禁用标志，`0` 表示启用，`1` 表示禁用。    |
| `CONTENT_DESC`          | 业务内容描述。                             |
| `BUSINESS_FORM`         | 业务表单相关信息，可能是表单结构或类型。      |
| `APPLY_TYPE`            | 申请类型标志。                             |
| `CUSTOM_FORM_LABEL`     | 自定义表单标签。                           |
| `FORM_COMPONENT_PATH`   | 表单组件路径。                             |
| `SYSTEM_NO`             | 系统编号，标识该业务所属的系统。             |
| `BUSINESS_AREA`         | 业务范围描述。                             |
| `PINYIN`                | 业务名称拼音，便于搜索或索引。               |
| `DRAFT_ONLY_OPEN_PARTS` | 草稿部分信息。                             |
| `USE_USERS`             | 业务适用用户信息。                         |
| `USE_ORGS`              | 业务适用组织信息。                         |
| `USE_GROUPS`            | 业务适用群组信息。                         |
| `USE_ROLES`             | 业务适用角色信息。                         |

### 表 `XYCS.FLOW_BUSINESS` 简介

#### **表用途**
`FLOW_BUSINESS` 表用于记录和管理与业务流程相关的核心信息，支持各类业务操作，包括公文管理、日常办公、审批管理等。表中存储的信息涵盖业务类型、编号、名称、适用范围以及表单配置路径等，为流程管理系统提供基础数据支持。

---

#### **表结构**

| 字段名                  | 描述                                       |
|--------------------------|--------------------------------------------|
| `ID`                    | 主键，唯一标识每条记录。                    |
| `UNIT_NO`               | 单位编号。                                 |
| `SORT_NO`               | 排序编号，用于控制显示顺序。                 |
| `BUSINESS_CATE`         | 业务类别，描述业务的分类，例如公文管理。      |
| `BUSINESS_NO`           | 业务编号，用于区分具体业务。                 |
| `BUSINESS_NAME`         | 业务名称，描述该业务的具体用途。             |
| `IS_DISABLE`            | 是否禁用标志，`0` 表示启用，`1` 表示禁用。    |
| `CONTENT_DESC`          | 业务内容的详细描述。                       |
| `BUSINESS_FORM`         | 业务表单信息，可能是表单的结构或类型定义。    |
| `APPLY_TYPE`            | 申请类型标志，用于区分业务申请的模式。        |
| `CUSTOM_FORM_LABEL`     | 自定义表单标签，便于业务的个性化管理。         |
| `FORM_COMPONENT_PATH`   | 表单组件路径，用于关联具体的表单实现。         |
| `SYSTEM_NO`             | 系统编号，标识该业务所属的系统模块。          |
| `BUSINESS_AREA`         | 业务范围，描述业务覆盖的场景或区域。          |
| `PINYIN`                | 业务名称拼音，便于搜索或索引。               |
| `DRAFT_ONLY_OPEN_PARTS` | 草稿部分信息，用于保存未完成的业务片段。       |
| `USE_USERS`             | 适用用户列表，存储业务适用的具体用户。         |
| `USE_ORGS`              | 适用组织列表，存储业务适用的组织信息。         |
| `USE_GROUPS`            | 适用群组列表，存储业务适用的群组信息。         |
| `USE_ROLES`             | 适用角色列表，存储业务适用的角色信息。         |
